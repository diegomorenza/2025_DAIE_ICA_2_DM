# Data Analytics for Immersive Environments – ICA Part C

This repository contains the Quarto Dashboard with Shiny developed for
Part C of the individual assignment.

## Project Description

The project is an interactive dashboard built using **Quarto Dashboards**
and **Shiny**. It allows users to explore virtual production data stored
in an SQLite database through tables and interactive plots.

## Dashboard Features

- **Data View**
  - Displays the contents of four database tables
  - Tables include pagination for easier navigation

- **Plot View**
  - Interactive plots based on production data
  - User input controls allow simple data exploration

## How to Run the Dashboard

1. Open the project in **RStudio** by double-clicking the `.Rproj` file.
2. Ensure required R packages are installed.
3. Open `index.qmd`.
4. Click **Render** or **Run Document** to launch the dashboard.

## Main File

- `index.qmd` – Entry point for the Quarto Dashboard
